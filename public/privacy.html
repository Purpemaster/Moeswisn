Allgemeine Informationen

Diese Website dient der Darstellung von Blockchain-basierten Inhalten und verfolgt keine kommerziellen Zwecke. Es werden keine personenbezogenen Daten direkt vom Website-Betreiber erhoben, keine Kontaktformulare angeboten und keine Cookies gesetzt.

Hosting & Sicherheit

Die Seite wird über GitHub Pages gehostet. Zum Schutz vor Angriffen und zur Optimierung der Auslieferung wird Cloudflare eingesetzt. Dabei kann es zur Verarbeitung von IP-Adressen kommen.

Spendenanzeige

Die Spendenfunktion zeigt öffentlich zugängliche Informationen einer Solana-Wallet an. Diese Wallet gehört nicht dem Betreiber der Website. Es findet keine Verknüpfung zwischen Spendern und Nutzern statt. Es werden keine personenbezogenen Daten gespeichert.

Externe APIs

Die Seite nutzt APIs von Drittanbietern, um aktuelle Spendenbeträge und Kursdaten anzuzeigen. Dabei wird Ihre IP-Adresse an folgende Anbieter übermittelt:

  • Helius API – Privacy
  • CoinGecko – Privacy
  • GeckoTerminal – Privacy

Radio-Streams

Beim Aktivieren eines Radio-Senders wird eine Verbindung zu Drittanbietern aufgebaut (z. B. laut.fm). Ihre IP-Adresse wird an deren Server übertragen. Diese Funktion ist optional und erfolgt nur durch aktive Nutzung.

Social Media

General Information

This website does not collect personal data. It is a non-commercial community project for displaying blockchain-related information. No tracking, cookies, or analytics are used.

Hosting & Security

This website is hosted via GitHub Pages and delivered securely using Cloudflare. IP addresses may be processed for performance and security purposes.

Donation Tracking

The site displays public blockchain information of a Solana wallet. This wallet is not operated by the site owner. No donor information is stored or processed.

External APIs

To show current token prices and balances, the site queries third-party APIs. Your IP address may be transmitted to:

  • Helius API – Privacy Policy
  • CoinGecko – Privacy Policy
  • GeckoTerminal – Privacy Policy

Radio Streams

When activating the radio feature, audio content is streamed from external providers (e.g., laut.fm). This transfers your IP address to their servers. This is optional and user-triggered.
