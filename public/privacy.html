Datenschutzerklärung

Diese Website erhebt und speichert keine personenbezogenen Daten. Es werden keine Cookies verwendet und keine Nutzerprofile erstellt.

Externe Inhalte

  • Cloudflare: Die Website wird über Cloudflare ausgeliefert, das technische Zugriffsdaten (z. B. IP-Adresse) verarbeitet.
  • GitHub Pages: Das Hosting erfolgt auf GitHub Pages (USA), technisch bedingte Zugriffsdaten werden dort verarbeitet.
  • Radio-Streams: Beim Aktivieren der Radiofunktion werden Audiodaten von externen Servern geladen. Es gelten die Datenschutzrichtlinien der jeweiligen Anbieter.
  • Social Media: Icons verlinken auf externe Plattformen (Instagram, X, TikTok etc.). Erst beim Klick werden Daten übertragen.
  • Solana Blockchain: Spenden erfolgen öffentlich über Solana-Wallets. Diese Transaktionen sind transparent, aber pseudonym.

Für technische Sicherheit setzen wir auf TLS-Verschlüsselung, keine Analyse-Tools, keine Werbung, keine Cookies.
